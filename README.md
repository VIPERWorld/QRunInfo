*QRunInfo
==============================
QRunInfo 是从Qt Installer Framework中提取出来的一些系统的信息获取和判断操作的封装。

================================
环境：Qt 5.0+ <br\>
系统:win,linux,mac、、、
=========================================
KDLockFile : 文件锁定和解锁 <br\>
KDRunOnceChecker : 检查程序是否是在运行中 ,或者文件有没有被锁定<br\>
QRunInfo : 获取系统内存信息，检索正在运行的进程，杀死进程，判断是否是本地文件 <br\>
AdminAuthorization : 判断当前程序是否是在管理员模式下运行，和以管理员程序开启启动程序 <br\>
・・・・・・
==============================================
注：<br\>
AdminAuthorization 在win下测试通过，其他类未测试，编译win下无误，其他平台未详细测试，请帮忙测试或者修正。
